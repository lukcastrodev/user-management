# Setup

- run *npm i -g @angular/cli*
- run *npm i*
- create a *db.json* file in the root of app
- add this content to *db.json*
```
  {
    "users": []
  }
```
- run *json-server --watch db.json*
- run *ng serve*

**Angular app**: *http://localhost:4200/*
<br/>
**Json-Server API**: *http://localhost:3000/*
